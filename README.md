#  User Task Dashboard

A modern and interactive User Task Management Dashboard built using HTML, CSS, and JavaScript.  
This application allows you to manage users and their tasks efficiently with features like filtering, progress tracking, and dark mode.

---

##  Features

- Add and manage multiple users  
- Create, edit, and delete tasks  
- Track task progress (0–100%)  
- Deadline tracking:
  - Overdue  
  - Due Today  
  - Upcoming  
- Search users and tasks  
- Filter tasks by:
  - Priority (High, Medium, Low)  
  - Status (Completed, In Progress, Overdue)  
- Light/Dark theme toggle  

---

##  Real-time Statistics

- Total Users  
- Total Tasks  
- Completed Tasks  

---

##  Tech Stack

- HTML5 – Structure  
- CSS3 – Styling (Custom properties, Flexbox, Grid)  
- JavaScript (Vanilla JS) – Logic & Interactivity  

---

##  How to Use

- Click "Add User" to create a user  
- Click "Add New Task" under a user  

### Fill in:

- Task Name  
- Priority  
- Category  
- Deadline  
- Progress  

- Use search and filters to manage tasks easily  
- Toggle between Dark/Light mode
